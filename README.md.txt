# Human Distress & Environmental Sound Detection

An audio classification system that identifies:

- Distress Sounds
- Alert Sounds
- Ambient Sounds

## Dataset
ESC-50

## Features
- MFCC
- Audio preprocessing

## Model
Random Forest

## Results
- Accuracy: 88.75%
- Distress Recall: 88%
